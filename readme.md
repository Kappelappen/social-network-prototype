# Social Network Prototype

A demonstration social networking platform built with PHP and MySQL.

## Purpose

This project explores the technical foundations behind digital communities:
user accounts, profiles, communication, and interaction between members.

## Built With

- PHP 8
- MySQL
- PDO
- HTML5
- CSS3
- Object-Oriented Programming

## Features

- User registration
- Authentication
- Profile management
- Profile images
- Member directory
- Comment system
- Pagination
- Database installation

## Future Vision

This prototype represents an exploration of how technology can 
create meaningful digital spaces where people can connect and 
share their stories.
